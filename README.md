# CloneTube - A YouTube UI Clone

CloneTube is a simple UI clone of YouTube built using HTML and CSS. This project mimics the basic structure and layout of YouTube, featuring a header, sidebar, and video previews. It focuses on creating a responsive and interactive user interface similar to YouTube.

## Features

- **Navigation Bar**: Includes a search bar, voice search, and user account options.
- **Sidebar**: Provides quick access to Home, Explore, Subscriptions, Library, and more.
- **Video Previews**: Displays video thumbnails, titles, channels, views, and durations in a structured grid layout.
- **Responsive Design**: Optimized layout with a grid system that adapts to different screen sizes.
- **Hover Effects**: Interactive hover effects on sidebar and header elements.
- **Fixed Header & Sidebar**: Ensures easy navigation even when scrolling.

## Technologies Used

- **HTML**: For structuring the webpage.
- **CSS**: Used for styling and layout design.
